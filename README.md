# E-Commerce_Fraud_Analytics
E-commerce fraud review analytics using Python, Excel, and Tableau.
## Project Overview

This project analyzes product reviews to detect fraudulent or suspicious reviews in e-commerce platforms.

## Tools Used

* Python (Data Cleaning & Sentiment Analysis)
* Excel (Initial Data Cleaning)
* Tableau (Interactive Dashboard)
* GitHub (Project Hosting)

## Features

* Fraud Score Calculation
* Sentiment Analysis using TextBlob
* Fraud Probability Scoring
* Interactive Fraud Analytics Dashboard

## Dashboard Insights

* Fraud vs Genuine Reviews
* Fraud Risk Distribution
* Sentiment Analysis of Reviews
* Verified vs Non-Verified Purchase Fraud

## Dataset

The dataset contains product reviews with rating, review text, verification status, and fraud indicators.
